📚 Book Management API Backend

Welcome to your Spring Boot Book Management API!

🚀 API Endpoints

GET /api/books
Get all books
POST /api/books
Add a new book
PUT /api/books/{id}
Update an existing book
DELETE /api/books/{id}
Delete a book
